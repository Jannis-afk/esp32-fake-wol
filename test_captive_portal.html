Manual Testing

To test the captive portal manually:

  1. Connect to the "ESP32-PC-Controller" WiFi network
  2. Open any website in your browser (e.g., google.com)
  3. You should be redirected to the ESP32 configuration page
  4. If not working, try visiting http://192.168.4.1 directly
